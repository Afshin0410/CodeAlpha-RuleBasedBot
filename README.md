# CodeAlpha-RuleBasedBot 🤖

An improved rule-based chatbot built in Python with keyword matching.

## Features
- Responds to greetings, questions, thanks, and farewells
- **Keyword matching** — understands variations like "heyy", "thanks a lot"
- Handles unknown inputs gracefully
- Runs in a continuous loop until the user types 'bye'

## How to Run
1. Make sure Python is installed
2. Clone this repository or download ChatBot.py
3. Run: python ChatBot.py

## Sample Output
Greeting with pleasure! Bot here.
You: hello
Roxx: Hi there!
You: thanks a lot
Roxx: You're welcome!
You: what's your name?
Roxx: I'm a simple chatbot, nice to meet you!
You: bye then
Roxx: Goodbye!

## Concepts Used
- Functions
- While loops
- If-elif statements
- String methods (.lower(), .strip())
- Keyword matching with in operator
